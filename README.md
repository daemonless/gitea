## Deployment

### Podman Compose

```yaml
services:
  gitea:
    image: ghcr.io/daemonless/gitea:latest
    container_name: gitea
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/gitea:/config"
    ports:
      - 3000:3000
      - 2222:2222
    restart: unless-stopped
```
